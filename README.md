## Entwicklung

```bash
# Dependencies installieren
npm install

# Development Server starten

```

Die Seite ist dann unter [http://localhost:3000](http://localhost:3000) erreichbar.

## Build

```bash
# Production Build erstellen
npm run build

# Production Server starten
npm start
```
